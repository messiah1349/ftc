Для запуска нового расчета нужно запустить в jupyter notebook файл FinTechCapital.ipynb. После чего в папке results появятся 6 папок с текущей датой: 3 флага дефолтности (30, 60, 90 дней) для выборки из всех клиентов и только для повторных клиентов.
В каждой папке есть папка pictures, в которой находятся картинки с распределениями переменных и связью их с целевой переменной, а также папка roc_auc с картинами рок-кривых.
Также в каждой папке есть папка model_output c файлом woeOutput.xlsx внутри нее. В файле 4 листа:
 - woeOut - содержит список всех переменных, включенных в модель, разбитых на бакеты.
 - woeProd - содержит необходимую информацию, если потребуется внедрять модель 
 - intercept - свободный член модели
 - information_value - содержит список всех переменных и их information value, отсортированных по убыванию IV 

В папке utils содержатся питон-файлы с библиотеками:
    fin_tech_capital_functions.py - файл с функциями преобразований переменных FinTechCapital
    scoringfunctions.py , woe.py - файлы с общими функциями для преобразований woe 

В папке sql лежит скрипт выгружающий все переменные для задачи
